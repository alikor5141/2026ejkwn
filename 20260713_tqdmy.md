<h2>正文</h2>
<p>Rubrik公司周二在Rubrik FORWARD大会上宣布，六家全球系统集成商将为企业客户部署其专为Anthropic Claude Code打造的Rubrik Agent Cloud平台。</p>
<p>加入“沙漏计划”的合作伙伴包括高知特、德勤、LTM、HCL科技、NTT Data以及威普罗，这些集成商将把该平台整合至各自的网络安全与数字化转型服务体系中。</p>
<p>Rubrik Agent Cloud为自主AI系统提供监控与治理能力，主要功能包括支持回滚操作的“代理回溯”功能、代码库恢复，以及针对AI代理行为控制的配置备份。</p>
<p>Rubrik首席解决方案官Alok Agrawal表示：“随着全球企业采用Claude Code，系统集成商正面临来自首席信息安全官和工程负责人的共同追问：如何在允许AI代理编写和部署代码的同时，避免引入灾难性的新风险？</p>
<p>”
Rubrik Zero Labs的数据显示，86%的网络安全负责人预计，AI代理将在未来一年内超越其组织现有的安全防护能力。</p>
<p>此次发布的平台旨在解决这一问题，为企业提供运行时安全防护、行为围栏以及爆炸半径控制等多层保护机制。</p>
<p>在合作细节方面，高知特将其整合至面向企业客户的Neuro AI平台中；德勤则将此次整合纳入其Ascend服务交付平台；HCL科技利用其VERITY Frontier AI韧性框架帮助客户实现安全的AI规模化部署；威普罗则结合其WINGS自动化运营框架和WEGA AI编排框架，将韧性嵌入代理生命周期。</p>
<p>Rubrik推出的“代理韧性合作伙伴计划”将为系统集成商提供技术认证，并开放与Rubrik工程团队的直接对接渠道。</p>
<p>面向Claude Code的Rubrik Agent Cloud现已正式上线，Rubrik将自身定位于数据保护、网络韧性与企业AI应用三大领域的交汇点。</p>
